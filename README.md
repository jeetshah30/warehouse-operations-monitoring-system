# Warehouse Operations Monitoring System

## Overview
A warehouse analytics platform built using Python, PostgreSQL, SQLAlchemy, and Streamlit.

## Features
- Inventory Monitoring
- Low Stock Alerts
- PostgreSQL ETL Pipeline
- SQL KPI Reporting
- Interactive Dashboard

## Tech Stack
- Python
- PostgreSQL
- SQLAlchemy
- Pandas
- Streamlit
- Git & GitHub

## KPIs
- Total Products
- Total Inventory Units
- Low Stock Products

## Project Architecture

CSV Files
↓
Python ETL
↓
PostgreSQL
↓
SQL Analytics
↓
Streamlit Dashboard
